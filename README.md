Unemployment Analysis with Python | Personal Data Science Project
(2019 – 2020)

Melakukan analisis data science end-to-end terhadap 1.000+ observasi tingkat pengangguran India (2019–2020) menggunakan Python, Pandas, NumPy, Matplotlib, dan Seaborn.
Membersihkan dan mentransformasi dua dataset mentah (missing values, duplikasi, standardisasi format tanggal) untuk memastikan keandalan analisis.
Melakukan exploratory data analysis melalui statistik deskriptif dan 10+ visualisasi data untuk mengidentifikasi tren, distribusi, dan pola regional.
Mengukur dampak kuantitatif pandemi Covid-19 terhadap unemployment rate menggunakan before-after comparison dan deteksi outlier berbasis IQR, mengungkap kenaikan 86,9% pada periode puncak lockdown.
Menyusun 10+ insight strategis dan rekomendasi kebijakan ekonomi-sosial berbasis data, didokumentasikan dalam laporan akademik dan repository GitHub yang reproducible.
